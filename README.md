# 후니의 잡동사니

## 개요
"후니의 잡동사니"는 제가 혼자 쓰려고 만든 기능들을 모아둔 Chrome 확장 프로그램입니다.

## 기능
- **광고 허용 안내 팝업 제거 (a.k.a AD Blocker Blocker Blocker..?)**: 웹 페이지에 나타나는 특정 클래스의 광고 허용 안내 팝업 및 관련 요소를 자동으로 제거하거나 숨깁니다.

  ↑ 이런거 차단합니다..
<br/>

- **코드 리뷰 요청 자동화**: GitHub PR 페이지에서 버튼 한 번으로 미리 설정된 리뷰어들에게 자동으로 리뷰를 요청합니다.
  - 레포별로 리뷰어를 다르게 설정할 수 있습니다.
  - 매번 리뷰어를 검색하고 클릭하는 귀찮음을 덜어줍니다. 제 소중한 5초를 아끼기 위한 기능입니다.
  - 설정 페이지에서 레포별로 코드 리뷰 자동화 규칙을 관리할 수 있습니다.

## 설치 및 사용법
### 설치
1. 이 저장소를 클론하거나 ZIP 파일로 다운로드합니다.
2. Chrome 브라우저를 엽니다.
3. 주소창에 `chrome://extensions`를 입력하고 엔터 키를 누릅니다.
4. 오른쪽 상단의 "개발자 모드" 토글을 활성화합니다.
5. "압축 해제된 확장 프로그램을 로드합니다." 버튼을 클릭합니다.
6. 클론하거나 다운로드한 저장소 폴더를 선택합니다.
7. 확장 프로그램이 설치되면 Chrome 툴바에 "후니의 잡동사니" 아이콘이 나타납니다.

### 사용법
1. 확장 프로그램 아이콘을 클릭하여 팝업을 엽니다.
2. "광고 허용 안내 끄기" 토글 스위치를 사용하여 기능을 활성화하거나 비활성화할 수 있습니다.
   - **활성화**: 웹 페이지의 광고 허용 안내 팝업이 자동으로 제거되고 스크롤이 복원됩니다.
   - **비활성화**: 확장 프로그램의 기능이 일시 중지됩니다. (이미 제거된 요소는 되돌려지지 않을 수 있습니다.)
3. 설정은 자동으로 저장되며, 브라우저를 닫았다 열어도 유지됩니다.
